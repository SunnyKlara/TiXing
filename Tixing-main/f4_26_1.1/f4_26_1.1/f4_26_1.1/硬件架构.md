## 1. 系统概述

RideWind 是一款智能 LED 风扇控制系统，硬件端基于 STM32F405 微控制器，通过 JDY-08 蓝牙模块与 Flutter APP 进行双向通信。

| 风扇控制 | PWM调速 (TIM3, 1kHz) |
| 雾化器 | GPIO开关控制 (PB8) |

## 4. 核心功能模块

### 4.1 蓝牙通信模块

**文件**: `Core/Src/rx.c`, `Core/Inc/rx.h`

**通信参数**:
- UART接口: UART2
- 波特率: 115200
- 数据格式: 8N1
- 接收方式: 中断接收
- 缓冲区大小: 200字节
- 超时判定: 50ms无新数据

**接收流程**:
```c
1. UART2中断接收 (HAL_UART_RxCpltCallback)
   ↓
2. 字节存入缓冲区 (ble_rx_buffer)
   ↓
3. 超时检测 (50ms无新数据)
   ↓
4. 协议解析 (Protocol_Process)
   ↓
5. 命令执行 (CMD_SetFanSpeed等)
   ↓
6. 响应发送 (BLE_SendString)
```

**关键函数**:
```c
void BLE_Process(void);                    // 主循环调用，处理接收数据
void BLE_SendString(const char* str);      // 发送字符串到APP
void BLE_ReportKnobDelta(int16_t delta);   // 上报旋钮增量
void BLE_ReportButtonEvent(const char* type, const char* action);  // 上报按钮事件
```

### 4.2 风扇控制模块

**文件**: `Core/Src/xuanniu.c`, `Core/Src/protocol.c`

**硬件配置**:
- 控制引脚: PC6 (TIM3_CH1)
- PWM频率: 1kHz
- PWM分辨率: 0-999 (Period=1000-1)
- 控制范围: 0-100%

**PWM映射公式**:
```c
// protocol.c
pwm_value = speed_percent * 10;  // 0-100 → 0-1000
if(pwm_value > 999) pwm_value = 999;
__HAL_TIM_SetCompare(&htim3, TIM_CHANNEL_1, pwm_value);
```

**控制接口**:
```c
void CMD_SetFanSpeed(uint8_t percent);  // 设置风扇速度 (0-100%)
uint8_t CMD_GetFanSpeed(void);          // 获取当前速度
```

### 4.3 LED控制模块

**文件**: `Core/Src/ws2812.c`, `Core/Inc/ws2812.h`

**硬件配置**:
- LED类型: WS2812B RGB灯珠
- 灯带数量: 4条
- 控制方式: PWM + DMA
- 颜色格式: RGB (0-255)

**灯带定义**:

|---------|---------|--------|---------|
| 1 | M (中间) | Main | RGB(150, 20, 0) 深橙红 |
| 2 | L (左侧) | left | RGB(255, 0, 0) 纯红 |
| 3 | R (右侧) | right | RGB(33, 126, 222) 蓝色 |
| 4 | B (后部) | tail | RGB(255, 0, 0) 纯红 |

**控制接口**:
```c

void WS2812B_Update(uint8_t strip);
void WS2812B_SetBrightness(uint8_t brightness);  // 全局亮度 0-100
```

**颜色参考** (xuanniu.h):

### 4.4 音频播放模块

**文件**: `Core/Src/audio_player.c`, `Core/Src/engine_audio.c`

**硬件配置**:

### 4.6 旋转编码器模块

**文件**: `Core/Src/xuanniu.c`, `Core/Inc/xuanniu.h`

**硬件配置**:
- 编码器接口: TIM1 (编码器模式)
- A相引脚: PE9 (TIM1_CH1)
- B相引脚: PE11 (TIM1_CH2)
- 按键检测: GPIO中断

**功能特性**:
- 旋转检测: 增量式编码器

**按键事件定义**:

| 三击 | 300ms内三次点击 | `BTN:KNOB:TRIPLE\n` |
| 长按 | 按下>1000ms | `BTN:KNOB:LONG\n` |

**控制接口**:

## 5. 通信协议规范

### 5.1 协议格式

采用文本协议，以换行符 `\n` 作为命令结束符。

**命令格式**: `CMD:PARAM\n`
**响应格式**: `OK:CMD:PARAM\r\n` 或 `ERR:message\r\n`

### 5.2 APP → 硬件 命令表

| 命令 | 参数 | 功能 | 示例 |
|------|------|------|------|
| `FAN:xx` | 0-100 | 设置风扇速度 | `FAN:50\n` |
| `GET:FAN` | 无 | 查询风扇速度 | `GET:FAN\n` |
| `SPEED:xxx` | 0-340 | 同步运行速度 | `SPEED:120\n` |
| `WUHUA:x` | 0/1 | 设置雾化器 | `WUHUA:1\n` |
| `GET:WUHUA` | 无 | 查询雾化器 | `GET:WUHUA\n` |
| `LED:s:r:g:b` | s=1-4, rgb=0-255 | 设置LED颜色 | `LED:1:255:0:0\n` |
| `PRESET:x` | 1-12 | 设置LED预设 | `PRESET:5\n` |
| `BRIGHT:xx` | 0-100 | 设置全局亮度 | `BRIGHT:80\n` |
| `GET:BRIGHT` | 无 | 查询亮度 | `GET:BRIGHT\n` |
| `UI:x` | 0-4 | 设置硬件UI | `UI:1\n` |
| `GET:UI` | 无 | 查询当前UI | `GET:UI\n` |
| `LCD:x` | 0/1 | 开/关LCD | `LCD:1\n` |
| `UNIT:x` | 0/1 | 设置单位 | `UNIT:0\n` |
| `THROTTLE:x` | 0/1 | 油门模式 | `THROTTLE:1\n` |
| `AUDIO:PLAY:xx` | 0-14 | 播放音频 | `AUDIO:PLAY:0\n` |
| `AUDIO:STOP` | 无 | 停止播放 | `AUDIO:STOP\n` |
| `AUDIO:PAUSE` | 无 | 暂停播放 | `AUDIO:PAUSE\n` |
| `AUDIO:RESUME` | 无 | 继续播放 | `AUDIO:RESUME\n` |
| `AUDIO:VOL:xx` | 0-100 | 设置音量 | `AUDIO:VOL:80\n` |
| `AUDIO:NEXT` | 无 | 下一首 | `AUDIO:NEXT\n` |
| `AUDIO:PREV` | 无 | 上一首 | `AUDIO:PREV\n` |
| `GET:AUDIO` | 无 | 查询音频状态 | `GET:AUDIO\n` |
| `GET:ALL` | 无 | 查询所有状态 | `GET:ALL\n` |

### 5.3 硬件 → APP 响应/上报表

| 响应/上报 | 格式 | 说明 |
|----------|------|------|
| 风扇状态 | `FAN:50\r\n` | 当前速度50% |
| 雾化器状态 | `WUHUA:1\r\n` | 1=开启, 0=关闭 |
| 亮度状态 | `BRIGHT:80\r\n` | 当前亮度80% |
| UI状态 | `UI:1\r\n` | 当前UI界面 |
| 音频状态 | `AUDIO:PLAYING:80:0:15\r\n` | 状态:音量:当前:总数 |
| 所有状态 | `STATUS:FAN:50:WUHUA:1:BRIGHT:80\r\n` | 批量状态 |
| 速度报告 | `SPEED_REPORT:120:0\n` | 速度:单位(0=km/h) |
| 油门报告 | `THROTTLE_REPORT:1\n` | 1=开启, 0=关闭 |
| 单位报告 | `UNIT_REPORT:0\n` | 0=km/h, 1=mph |
| 预设报告 | `PRESET_REPORT:5\n` | 当前预设索引 |
| 旋钮增量 | `KNOB:5\n` | 正=顺时针, 负=逆时针 |
| 按钮事件 | `BTN:KNOB:CLICK\n` | 类型:动作 |
| 传感器 | `SENSOR:TEMP:45\n` | 类型:数值 |
| 操作成功 | `OK:CMD:PARAM\r\n` | 命令执行成功 |
| 操作失败 | `ERR:message\r\n` | 错误信息 |

## 6. 双向通信架构

### 6.1 通信流程图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         双向通信数据流                                       │
└─────────────────────────────────────────────────────────────────────────────┘

                    APP 控制命令 (APP → 硬件)
┌─────────────┐    ─────────────────────────────────▶    ┌─────────────┐
│             │    FAN:50\n, LED:1:255:0:0\n, WUHUA:1\n  │             │
│   Flutter   │                                          │   STM32     │
│    APP      │    ◀─────────────────────────────────    │   MCU       │
│             │    硬件状态上报 (硬件 → APP)              │             │
└─────────────┘    SPEED_REPORT:120:0\n, KNOB:5\n        └─────────────┘
       │                                                        │
       │                                                        │
       ▼                                                        ▼
┌─────────────┐                                          ┌─────────────┐
│ BLEService  │    ◀────── BLE 透传通道 ──────▶         │  UART2      │
│ Protocol    │           (20字节/包)                    │  rx.c       │
│ Service     │                                          │  protocol.c │
└─────────────┘                                          └─────────────┘
```

### 6.2 主动上报机制

硬件端在以下情况主动上报数据到APP：

| 触发条件 | 上报内容 | 用途 |
|---------|---------|------|
| 旋钮旋转 | `KNOB:delta\n` | APP同步滚轮位置 |
| 旋钮按键 | `BTN:KNOB:action\n` | APP响应按键事件 |
| 速度变化 | `SPEED_REPORT:speed:unit\n` | APP同步速度显示 |
| 油门模式切换 | `THROTTLE_REPORT:state\n` | APP同步油门状态 |
| 单位切换 | `UNIT_REPORT:unit\n` | APP同步单位显示 |
| 预设切换 | `PRESET_REPORT:index\n` | APP同步预设选择 |

### 6.3 同步查询机制

APP可以发送 `GET:xxx` 命令查询硬件状态，硬件响应对应数据：

```
APP发送: GET:FAN\n
硬件响应: FAN:50\r\n

APP发送: GET:ALL\n
硬件响应: STATUS:FAN:50:WUHUA:1:BRIGHT:80\r\n
```

## 8. 主循环流程

```c
// main.c 主循环
int main(void) {
    // 1. 系统初始化
    HAL_Init();
    SystemClock_Config();
    
    // 2. 外设初始化
    MX_GPIO_Init();
    MX_DMA_Init();
    MX_USART1_UART_Init();  // 调试串口
    MX_USART2_UART_Init();  // 蓝牙串口
    MX_TIM1_Init();         // 编码器
    MX_TIM3_Init();         // 风扇PWM
    MX_SPI1_Init();         // LCD
    MX_SPI2_Init();         // Flash
    MX_SPI3_Init();         // VS1003
    
    // 3. 模块初始化
    WS2812B_Init();
    LCD_Init();
    AudioPlayer_Init();
    BLE_Init();
    
    // 4. 启动PWM和编码器
    HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);
    HAL_TIM_Encoder_Start(&htim1, TIM_CHANNEL_ALL);
    
    // 5. 主循环
    while (1) {
        BLE_Process();      // 处理蓝牙数据
        Encoder();          // 处理编码器
        LCD_Update();       // 更新LCD显示
        AudioPlayer_Process(); // 音频处理
        
        HAL_Delay(1);       // 1ms循环周期
    }
}
```

## 9. 关键代码示例

### 9.1 蓝牙数据接收处理

```c
// rx.c
#define BLE_RX_BUFFER_SIZE 200
uint8_t ble_rx_buffer[BLE_RX_BUFFER_SIZE];
uint16_t ble_rx_index = 0;
uint32_t ble_rx_timeout = 0;

// UART接收中断回调
void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart) {
    if (huart == &huart2) {
        if (ble_rx_index < BLE_RX_BUFFER_SIZE - 1) {
            ble_rx_buffer[ble_rx_index++] = rx_byte;
        }
        ble_rx_timeout = HAL_GetTick();
        HAL_UART_Receive_IT(&huart2, &rx_byte, 1);
    }
}

// 主循环中调用
void BLE_Process(void) {
    // 50ms超时判定一包数据完成
    if (ble_rx_index > 0 && (HAL_GetTick() - ble_rx_timeout) > 50) {
        ble_rx_buffer[ble_rx_index] = '\0';
        Protocol_Process((char*)ble_rx_buffer);
        ble_rx_index = 0;
    }
}

// 发送字符串到APP
void BLE_SendString(const char* str) {
    HAL_UART_Transmit(&huart2, (uint8_t*)str, strlen(str), 100);
}
```

### 9.4 按钮事件检测

```c
// xuanniu.c
void Check_KnobButton(void) {
    static uint32_t press_time = 0;
    static uint8_t click_count = 0;
    static uint32_t last_click_time = 0;
    
    if (KNOB_BTN_PRESSED) {
        if (press_time == 0) {
            press_time = HAL_GetTick();
        }
    } else if (press_time > 0) {
        uint32_t duration = HAL_GetTick() - press_time;
        
        if (duration > 1000) {
            // 长按
            BLE_ReportButtonEvent("KNOB", "LONG");
        } else {
            // 短按，累计点击次数
            click_count++;
            last_click_time = HAL_GetTick();
        }
        press_time = 0;
    }
    
    // 300ms内无后续点击，上报点击事件
    if (click_count > 0 && (HAL_GetTick() - last_click_time) > 300) {
        if (click_count >= 3) {
            BLE_ReportButtonEvent("KNOB", "TRIPLE");
        } else if (click_count == 2) {
            BLE_ReportButtonEvent("KNOB", "DOUBLE");
        } else {
            BLE_ReportButtonEvent("KNOB", "CLICK");
        }
        click_count = 0;
    }
}

// rx.c
void BLE_ReportButtonEvent(const char* type, const char* action) {
    char buf[32];
    sprintf(buf, "BTN:%s:%s\n", type, action);
    BLE_SendString(buf);
}
```
