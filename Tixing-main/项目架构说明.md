# 项目架构说明

## 目录结构

```
📁 项目根目录/（需烧录到 Pico 的文件）
├── main.py              # 主控程序 - f4 风格界面状态机 + 交互逻辑 + 配置恢复
├── display.py           # 界面绘制模块 - 所有 draw_xxx_screen() 函数

├── st7789py.py          # ST7789 LCD 显示驱动（SPI通信 + 图形绘制）
├── lcdfont.py           # 8x8 ASCII 点阵字体库（ASCII 32-126）

📁 不需要烧录的文件/
├── test_config.py       # config.py 单元测试
├── test_effects.py      # effects.py 单元测试
├── .kiro/               # Kiro IDE spec 文件

```

## 界面状态机（f4 风格 ui/init_flag 双变量机制）

采用 f4 项目的 `ui`/`chu` 设计模式：
- `ui` 变量控制当前界面编号
- `init_flag` 控制一次性初始化（切换界面时置 True，绘制后重置）
- 避免每帧全屏重绘，只在状态变化时刷新

| 界面常量 | 编号 | 名称 | 功能 |
|---------|------|------|------|
| UI_LOGO | 0 | 开机动画 | Logo + COB 灯带交替闪烁渐亮 |
| UI_MENU | 1 | 菜单 | 6 选项水平滚动，圆点指示器 |
| UI_WINDSPEED | 2 | 风速控制 | 0-100% 进度条，支持油门模式 |
| UI_SMOKESPEED | 3 | 发烟控制 | 0-100% 进度条 |
| UI_AIRPUMP | 4 | 气泵控制 | 0-100% 进度条 |
| UI_PRESET | 5 | 灯光预设 | 12 种颜色预设 + 渐变模式 |
| UI_RGBCONTROL | 6 | RGB 调色 | 三层状态机：选灯带→选通道→调数值 |
| UI_BRIGHTNESS | 7 | 亮度调节 | 0-100% + 呼吸灯模式开关 |

## 各模块功能详解

### main.py — 主控程序

| 功能分类 | 具体内容 |
|---------|---------|
| 硬件初始化 | SPI(0) 总线、DC/CS/Reset 引脚、ST7789 实例化（rotation=1 横屏） |
| 状态机 | `ui`/`init_flag` 双变量，`switch_screen()` 切换界面 |
| 状态字典 | `get_state()` 构建完整状态传递给 display/effects |
| 交互分发 | 主循环按 `ui` 分发编码器和按键事件到各界面处理逻辑 |
| 硬件同步 | `update_hardware()` 将 UI 参数实时写入 PWM 外设 |
| 配置恢复 | 开机 `config.load_config()` 恢复上次设置 |
| 配置保存 | `save_current_config()` 双击返回菜单时批量保存 |
| 油门模式 | 风速界面三击进入，按住加速松开减速，旋转退出 |
| 效果调用 | 每帧调用 `effects_process()`（RGB 调色界面除外） |

### display.py — 界面绘制模块

| 功能分类 | 具体内容 |
|---------|---------|
| 颜色常量 | HIGHLIGHT(绿)、NORMAL(白)、GRAY、DARK、BACKGROUND(黑) |
| 布局参数 | 屏幕 284×76、字体 8×8、进度条/圆点位置参数 |
| 颜色预设 | `COLOR_PRESETS` 12 种预设（Cyber/Ice/Sunset/Racing 等） |
| 菜单标签 | `MODE_LABELS` = Wind/Smoke/AirPump/Preset/RGB/Bright |
| 绘制函数 | 8 个 `draw_xxx_screen(tft, state)` 函数，接收 tft 和状态字典 |
| 辅助函数 | `draw_circle()` 填充/空心圆 |

所有绘制函数：

| 函数 | 界面 | 特殊元素 |
|------|------|---------|
| `draw_logo_screen()` | 开机 Logo | 居中 "Hello World" |
| `draw_mode_select_screen()` | 菜单 | 居中文字 + 6 个圆点指示器 |
| `draw_windspeed_screen()` | 风速 | 进度条 + 百分比 + THR 油门指示 |
| `draw_smokespeed_screen()` | 发烟 | 进度条 + 百分比 |
| `draw_airpump_screen()` | 气泵 | 进度条 + 百分比 |
| `draw_preset_screen()` | 预设 | 编号 + 名称 + Grad 指示 + COB 颜色预览 |
| `draw_rgbcontrol_screen()` | RGB | 模式指示 + 灯带名 + R/G/B 通道值 |
| `draw_brightness_screen()` | 亮度 | 进度条 + Breath:ON/OFF 指示 |

### effects.py — 灯光效果引擎

| 功能分类 | 具体内容 |
|---------|---------|
| 优先级系统 | breathing > preset_gradient > normal |
| 主处理函数 | `effects_process(state, set_cob1_fn, set_cob2_fn)` 每帧调用 |
| 呼吸灯 | `_breathing_process()` — math.sin 正弦波，628 步/周期 ≈ 3 秒 |
| 渐变效果 | `_gradient_process()` — 12 预设间线性插值，100 步/段 |
| 状态重置 | `reset_effects()` — 进入 RGB 调色时调用，暂停效果 |
| 开机动画 | `startup_animation()` — Phase1: 3 次交替闪烁，Phase2: 线性渐亮 |

### config.py — 配置持久化

| 功能分类 | 具体内容 |
|---------|---------|
| 存储文件 | `settings.json`（JSON 格式，自动创建） |
| 加载 | `load_config()` — 文件不存在或损坏时用默认值，新 key 自动补全 |
| 保存 | `save_config()` — 仅 dirty=True 时写入，减少 Flash 擦写 |
| 读写 | `get(key)` / `set(key, value)` — set 自动标记 dirty |
| 脏标志 | `is_dirty()` / `mark_dirty()` |

配置项：

| Key | 类型 | 默认值 | 说明 |
|-----|------|--------|------|
| fan_speed | int | 0 | 风扇速度 0-100 |
| smoke_speed | int | 0 | 发烟速度 0-100 |
| air_pump_speed | int | 0 | 气泵速度 0-100 |
| cob1_rgb | list | [0,0,0] | COB1 颜色 0-255 |
| cob2_rgb | list | [0,0,0] | COB2 颜色 0-255 |
| brightness | int | 100 | 全局亮度 0-100 |
| color_preset | int | 1 | 预设索引 0-11 |
| led_mode | str | "normal" | normal/breathing/gradient |

### control.py — 外设控制

| 功能分类 | GPIO | 控制函数 |
|---------|------|---------|
| 主风扇 | GPIO6 | `set_fan_speed(0-100)` |
| 小风扇 | GPIO10 | `set_small_fan_speed(0-100)` |
| 气泵 | GPIO11 | `set_air_pump_speed(0-100)` |
| 发烟器 | GPIO12 | `set_smoke_generator_speed(0-100)` |
| COB1 灯带 | GPIO13/14/15 (R/G/B) | `set_cob1_rgb(r,g,b)` `set_cob1_single_color()` |
| COB2 灯带 | GPIO7/8/9 (R/G/B) | `set_cob2_rgb(r,g,b)` `set_cob2_single_color()` |
| 全局控制 | — | `stop_all_devices()` `set_all_cob_rgb()` |

所有控制函数参数范围 0-100（百分比），内部转换为 16 位 PWM 占空比。

### key.py — 编码器输入

| 功能分类 | 具体内容 |
|---------|---------|
| 硬件引脚 | GPIO19（按键）、GPIO20（A相）、GPIO21（B相） |
| 旋转识别 | A/B 相中断检测，非阻塞消抖（2ms 最小间隔） |
| 增量读取 | `get_encoder_delta()` — 原子读取累积增量并清零 |
| 旧接口 | `get_encoder_dir()` — 逐个消费 ±1（向后兼容） |
| 按键识别 | 消抖 20ms、长按 1000ms、多击间隔 300ms |
| 定时器优化 | 持久化 `_multi_click_timer` / `_long_press_timer`，避免竞争 |
| 事件类型 | `get_key_event()` → click1 / click2 / click3 / long_press / None |
| 直接读取 | `key_pin` 导出供油门模式直接读取按键物理状态 |

## 交互逻辑总览

| 操作 | 菜单界面 | 子界面（通用） | 风速界面（特殊） |
|------|---------|-------------|---------------|
| 旋转 | 切换选项（150ms 防跳） | 调节参数值 | 普通模式调速 / 油门模式退出 |
| 单击 | 进入选中的子界面 | 界面特定功能 | — |
| 双击 | — | 保存配置 + 返回菜单 | 保存 + 返回菜单 |
| 三击 | — | — | 进入油门模式 |
| 长按 | — | 预留 | — |

各子界面单击功能：
- 预设界面：切换渐变模式 ON/OFF
- RGB 界面：进入下一层（选灯带→选通道→调数值→回到选通道）
- 亮度界面：切换呼吸灯模式 ON/OFF

## 灯光效果优先级

```
┌─────────────────────────────────────────┐
│  effects_process() 每帧调用（10ms/帧）    │
│                                         │
│  if breath_mode:     → 呼吸灯（最高）     │
│  elif gradient_mode: → 预设渐变           │
│  else:               → normal（不处理）   │
│                                         │
│  进入 RGB 调色界面时：                     │
│    reset_effects() + 跳过 effects_process │
│    用户直接控制 COB 颜色                   │
└─────────────────────────────────────────┘
```

## 主循环执行顺序

```python
while True:
    encoder_delta = get_encoder_delta()   # 1. 统一读取编码器增量
    key_event = get_key_event()           # 2. 读取按键事件
    # 3. 按 ui 分发交互处理（菜单/风速/发烟/气泵/预设/RGB/亮度）
    update_display()                      # 4. 界面刷新（init_flag 控制）
    if ui != UI_RGBCONTROL:
        effects_process(state, ...)       # 5. 灯光效果处理
    sleep_ms(10)                          # 6. 10ms 帧间隔
```
