# 界面操控逻辑总结

## 界面概览

| UI编号 | 名称 | 功能 | 菜单位置 |
|--------|------|------|----------|
| UI0 | 开机动画 | 显示固定的启动画面 | 无（仅开机时） |
| UI1 | 调速界面 | 风扇速度控制、油门模式 | menu_selected=1 |
| UI2 | 颜色预设 | 14种LED颜色预设切换、流水灯 | menu_selected=2 |
| UI3 | RGB调色 | 自定义四条灯带RGB颜色 | menu_selected=3 |
| UI4 | 亮度调节 | LED亮度调节、呼吸灯模式 | menu_selected=4 |
| UI5 | 菜单界面 | 全屏滑动式菜单导航 | - |
| UI6 | Logo界面 | 展示用户上传的自定义图片 | menu_selected=5 |
| UI7 | 音量调节 | 音频音量调节 | menu_selected=6 |

---

## 各界面详细操控逻辑

### UI5 - 菜单界面（入口）
**功能：** 全屏滑动式菜单，选择进入其他界面

| 操作 | 功能 |
|------|------|
| 旋转 | 切换选中的界面（1-6循环） |
| 单击 | 进入选中的界面 |
| 双击 | 无反应 |
| 三击 | 无反应 |

**关键变量：** `menu_selected`（1-6）

---

### UI1 - 调速界面
**功能：** 控制风扇速度，支持油门模式

#### 普通模式（wuhuaqi_state ≠ 2）
| 操作 | 功能 |
|------|------|
| 旋转 | 调节速度（0-340 km/h） |
| 单击 | 切换单位（km/h ↔ mph） |
| 双击 | 返回菜单 |
| 三击 | 进入油门模式 |

#### 油门模式（wuhuaqi_state == 2）
| 操作 | 功能 |
|------|------|
| 按住 | 加速（速度递增） |
| 松开 | 减速（速度递减） |
| 旋转 | 退出油门模式 |
| 双击 | 无反应（不退出） |

**关键变量：** `Num`(0-100), `current_speed_kmh`(0-340), `wuhuaqi_state`, `speed_value`(单位)

**PWM控制：** `Num * 10` → TIM3_CH1/CH4

---

### UI2 - 颜色预设界面
**功能：** 14种LED颜色预设切换，支持流水灯模式

| 操作 | 功能 |
|------|------|
| 旋转 | 切换颜色预设（1-14循环）；流水灯模式下旋转会退出流水灯 |
| 单击 | 切换流水灯开关（deng_2or3） |
| 双击 | 保存颜色并返回菜单 |

**关键变量：** `deng_num`(1-14), `deng_2or3`(0=普通/1=流水灯), `pei_se_state`

**颜色同步：** 
- `_zhong`变量 → 临时编辑值
- `deng_update()` → 同步到实际变量并保存Flash
- `deng()` → 应用到LED

---

### UI3 - RGB调色界面
**功能：** 自定义四条灯带的RGB颜色（三层状态机）

#### 状态机模式
| ui3_mode | 状态 | 指示灯 |
|----------|------|--------|
| 0 | 选择灯带 | 红点 |
| 1 | 选择通道(R/G/B) | 绿点 |
| 2 | 调节数值(0-255) | 绿点+高亮 |

#### 操作逻辑
| 当前模式 | 旋转 | 单击 | 双击 |
|----------|------|------|------|
| 选灯带(0) | 切换灯带(M/L/R/B) | 进入选通道 | 保存并返回菜单 |
| 选通道(1) | 切换R/G/B | 进入调节 | 保存并返回菜单 |
| 调节(2) | 调节数值±2 | 退出到选通道 | 保存并返回菜单 |

**关键变量：** `ui3_mode`, `ui3_channel`(0=R/1=G/2=B), `name`(0-3灯带), `red1_zhong`等

---

### UI4 - 亮度调节界面
**功能：** 调节LED整体亮度，支持呼吸灯模式

| 操作 | 功能 |
|------|------|
| 旋转 | 普通模式：调节亮度（0-100）；呼吸灯模式：切换颜色预设 |
| 单击 | 切换呼吸灯模式（红点↔绿点） |
| 双击 | 保存亮度并返回菜单（呼吸灯保持运行） |

**关键变量：** `bright`(0-100), `breath_mode`(0=关/1=开), `breath_phase`, `breath_color_index`(1-14)

**呼吸灯持久化：** ✅ 已实现
- 退出UI4时 `breath_mode` 保持不变
- `Breath_Process()` 在主循环中持续运行呼吸效果
- 进入UI2或UI3时自动关闭呼吸灯（优先级：RGB调色 > 呼吸灯 > 流水灯）

---

### UI6 - Logo界面
**功能：** 展示用户从APP上传的自定义图片（支持3个槽位）

| 操作 | 功能 |
|------|------|
| 旋转 | 切换Logo槽位（多个有效时） |
| 单击 | 设为开机Logo（激活槽位） |
| 长按(≥2秒) | 删除当前槽位Logo |
| 双击 | 返回菜单 |

**关键变量：** `logo_view_slot`(0-2), `logo_slot_count`

**存储：** Flash外部存储，每个槽位115200字节

---

### UI7 - 音量调节界面
**功能：** 调节音频输出音量

| 操作 | 功能 |
|------|------|
| 旋转 | 调节音量（0-100） |
| 单击 | 切换调节状态（红点↔绿点） |
| 双击 | 返回菜单 |

**关键变量：** `volume`(0-100), `volume_state`

---

## 潜在的交叉逻辑问题及解决方案

### 0. LED效果优先级系统 ✅ 已实现
**优先级（从高到低）：**
1. **RGB调色 (UI3)** - 进入时关闭流水灯和呼吸灯
2. **呼吸灯 (UI4)** - 进入时关闭流水灯，退出后持续运行
3. **流水灯 (UI2)** - 最低优先级

**实现方式：**
- 进入UI2时：`deng_2or3 = 0; breath_mode = 0;`
- 进入UI3时：`deng_2or3 = 0; breath_mode = 0;`
- 进入UI4时：`deng_2or3 = 0;`（呼吸灯可在此界面开启）
- 主循环处理：`Breath_Process()` 在 `breath_mode==1` 且 `ui!=4` 时运行

### 1. LED颜色冲突
**问题：** 多个模块同时更新LED（流水灯、渐变、普通模式）
**现状：** 
- `Streamlight_Process()` - 流水灯
- `LED_GradientProcess()` - 渐变效果
- `deng()` - 普通模式

**解决方案：** ✅ 已修复
```c
// LED_GradientProcess() 开头添加
if (deng_2or3 == 1) return;  // 流水灯模式下跳过渐变
```

### 2. 颜色变量不同步
**问题：** `_zhong`变量和实际颜色变量不同步
**场景：** UI2选择预设后退出，进入UI1颜色不对

**解决方案：** ✅ 已修复
- `deng_ui2()` 同步设置 `red1_zhong`/`red4_zhong`
- UI2退出时调用 `deng_update()` + `deng()`

### 3. 油门模式状态残留
**问题：** 油门模式下切换界面可能导致状态混乱
**现状：** 油门模式下双击不返回菜单

**建议：** 
- 油门模式只能通过旋转退出
- 退出时强制重置 `Num=0`

### 4. 编码器增量累积
**问题：** 切换界面时残留的 `encoder_delta` 可能触发误操作
**现状：** 部分界面初始化时清零

**建议：** 统一在 `chu` 初始化时清零
```c
if(chu == X) {
    encoder_delta = 0;  // 清除残留增量
    // ...初始化代码
}
```

### 5. Flash写入时机
**问题：** 频繁写入Flash影响寿命
**现状：** 每次 `deng_update()` 都写Flash

**建议：** 
- 只在退出界面时写入
- 添加脏标志位，只有修改过才写入

### 6. PWM与界面状态
**问题：** `PWM()` 只在 `ui==1` 时更新
**影响：** 其他界面无法控制风扇

**建议：** 
- 蓝牙命令直接设置PWM（已实现）
- 或移除 `ui==1` 限制

### 7. 蓝牙命令与本地操作冲突
**问题：** APP和旋钮同时操作可能冲突
**现状：** 使用 `remote_active_tick` 防止反馈循环

**建议：** 
- 远程命令设置后短时间内忽略本地上报
- 使用 `last_reported_speed` 防止重复上报

## 全局状态变量一览

| 变量 | 类型 | 范围 | 用途 |
|------|------|------|------|
| `ui` | u8 | 0-7 | 当前界面 |
| `chu` | u8 | 0-7 | 界面初始化标志 |
| `menu_selected` | u8 | 1-6 | 菜单选中项 |
| `Num` | int16 | 0-100 | 风扇速度百分比 |
| `current_speed_kmh` | int16 | 0-340 | 绝对速度 |
| `wuhuaqi_state` | u8 | 0/1/2 | 雾化器状态(0关/1开/2油门) |
| `deng_num` | u8 | 1-14 | 颜色预设编号 |
| `deng_2or3` | u8 | 0/1 | 流水灯开关 |
| `bright` | int16 | 0-100 | LED亮度 |
| `volume` | u8 | 0-100 | 音频音量 |
| `ui3_mode` | u8 | 0-2 | RGB调色状态机 |
| `breath_mode` | u8 | 0/1 | 呼吸灯开关 |
