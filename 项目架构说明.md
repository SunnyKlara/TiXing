# 项目架构说明

## 目录结构

```
📁 项目根目录/
├── main.py              # 主控程序 - 界面状态机 + 硬件初始化 + 交互逻辑
├── st7789py.py          # ST7789 LCD 显示驱动（SPI通信 + 图形绘制）
├── lcdfont.py           # 8x8 ASCII 点阵字体库（ASCII 32-126）
├── control.py           # PWM 外设控制（风扇/气泵/发烟器/COB灯带）
└── key.py               # EC10 滚轮编码器输入（旋转 + 按键事件）
```

## 各模块功能详解

### main.py — 主控程序

| 功能分类 | 具体内容 |
|---------|---------|
| 硬件初始化 | SPI(0) 总线、DC/CS/Reset 引脚、ST7789 实例化（rotation=1 横屏） |
| 界面常量 | 屏幕尺寸 284×76、字体 8×8、各界面布局参数 |
| 界面绘制 | `draw_logo_screen()` — Hello World 开机界面 |
| | `draw_mode_select_screen()` — 模式选择（3个圆点指示器） |
| | `draw_windspeed_screen()` — 风速控制（进度条 + 百分比） |
| | `draw_smokespeed_screen()` — 发烟速度控制 |
| | `draw_rgbcontrol_screen()` — RGB 灯带控制（3通道独立调节） |
| 交互逻辑 | 旋钮切换模式/调节参数、单击进入/编辑、双击返回 |
| 硬件同步 | `update_hardware()` 将 UI 参数实时写入 PWM 外设 |

### st7789py.py — 显示驱动

| 功能分类 | 具体内容 |
|---------|---------|
| 初始化 | 硬件复位、软件复位、初始化命令序列、旋转配置 |
| 颜色 | `color565()` RGB→RGB565 编码、预定义颜色常量（BGR 修正） |
| 基础绘制 | `pixel()` `hline()` `vline()` `line()` `rect()` `fill_rect()` `fill()` |
| 文字渲染 | `text()` 8×8/16×16 字体绘制、`write()` 变宽字体绘制 |
| 位图 | `bitmap()` `pbitmap()` 调色板位图绘制 |
| 高级图形 | `polygon()` 多边形（支持旋转） |
| 滚动 | `vscrdef()` `vscsad()` 垂直滚动 |

### lcdfont.py — 字体库

| 功能分类 | 具体内容 |
|---------|---------|
| 字体规格 | 8×8 像素等宽点阵 |
| 字符范围 | ASCII 32（空格）到 126（~），共 95 个字符 |
| 数据格式 | 每字符 8 字节，逐行存储位图数据 |
| 覆盖内容 | 标点符号、数字 0-9、大写 A-Z、小写 a-z、特殊符号 |

### control.py — 外设控制

| 功能分类 | GPIO | 控制函数 |
|---------|------|---------|
| 主风扇 | GPIO6 | `set_fan_speed(0-100)` |
| 小风扇 | GPIO10 | `set_small_fan_speed(0-100)` |
| 气泵 | GPIO11 | `set_air_pump_speed(0-100)` |
| 发烟器 | GPIO12 | `set_smoke_generator_speed(0-100)` |
| COB1 灯带 | GPIO13/14/15 (R/G/B) | `set_cob1_rgb(r,g,b)` `set_cob1_single_color()` |
| COB2 灯带 | GPIO7/8/9 (R/G/B) | `set_cob2_rgb(r,g,b)` `set_cob2_single_color()` |
| 全局控制 | — | `stop_all_devices()` `set_all_cob_rgb()` |

### key.py — 编码器输入

| 功能分类 | 具体内容 |
|---------|---------|
| 硬件引脚 | GPIO19（按键）、GPIO20（A相）、GPIO21（B相） |
| 旋转识别 | A/B 相中断检测，`get_encoder_dir()` → +1/-1/0 |
| 按键识别 | 消抖 20ms、长按 1000ms、多击间隔 500ms |
| 事件类型 | `get_key_event()` → click1 / click2 / click3 / long_press / None |

│  GPIO9  ── COB2-B          │
